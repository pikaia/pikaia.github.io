# pikaia.github.io

Source for my blog on lesser-known facts and people from Singapore's history, built with Jekyll and hosted on GitHub Pages.

Live at: https://pikaia.github.io

## Adding a new post

1. Go to the `_posts` folder → **Add file → Create new file**
2. Name it `YYYY-MM-DD-your-title.md` (must follow this format)
3. Start the file with:
   ```
   ---
   layout: post
   title: "Your Post Title"
   date: YYYY-MM-DD
   categories: [history]
   ---
   ```
4. Write the post body in Markdown below the `---` block
5. Commit — the site rebuilds automatically in about a minute
